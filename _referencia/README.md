# 📁 Carpeta de Referencia — App Conciliación SDLs

Esta carpeta contiene los documentos de referencia del proyecto que son leídos por el asistente de IA (Antigravity) para mantener contexto en todas las conversaciones.

## Archivos disponibles

| Archivo | Descripción |
|---------|-------------|
| `PRD.md` | Product Requirements Document — descripción funcional del producto |
| `design_system.md` | Design System BIA Energy — tokens, colores, componentes CSS |

## 🚀 Cómo usarla

1. **PRD**: Abre `PRD.md` y pega o escribe el Product Requirements Document completo del proyecto. Incluye módulos, flujos, reglas de negocio y casos de uso.

2. **Design System**: `design_system.md` ya contiene los estilos actuales extraídos del proyecto. Puedes agregar lineamientos adicionales de marca.

## 📌 Convención

- Mantén los archivos **actualizados** cuando cambien los requerimientos.
- Puedes agregar más archivos `.md` en esta carpeta (ej. `arquitectura.md`, `reglas_negocio.md`).
- Al iniciar una conversación con el asistente, menciona "revisa `_referencia/`" para que tome el contexto.
